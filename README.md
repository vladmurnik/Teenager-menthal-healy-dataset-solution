# Teen Mental Health Analysis

## Project Overview

This project presents an exploratory analysis of the `Teen_Mental_Health_Dataset.csv` dataset. The purpose of the analysis is to investigate factors that may influence adolescent mental health and identify possible relationships between behavioral, social, and psychological variables.

The notebook combines data preprocessing, visualization, and analytical methods to extract meaningful insights and improve understanding of patterns in the dataset.

---

## Project Goals

The main goals of this project are:

- Explore the structure of the dataset
- Clean and preprocess the data
- Identify missing or inconsistent values
- Analyze relationships between different variables
- Visualize important patterns and trends
- Generate insights related to adolescent mental health factors

---

## Dataset Description

Dataset used:

**Teen_Mental_Health_Dataset.csv**

The dataset contains information associated with teenagers and factors that can potentially affect mental health, including:

- Social behavior indicators
- Lifestyle habits
- Psychological attributes
- Daily activity patterns
- Additional demographic or behavioral characteristics
